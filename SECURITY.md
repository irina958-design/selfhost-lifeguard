# Security Policy

## Supported version

Only the latest release is supported during the pilot.

## Report a vulnerability

Use GitHub's private **Report a vulnerability** form:

https://github.com/irina958-design/selfhost-lifeguard/security/advisories/new

Do not publish credentials, `.env` contents, personal file paths, database dumps, or vulnerability details in a public issue.

For ordinary non-sensitive bugs, open a regular GitHub issue with the Lifeguard version, operating system, Docker version, and redacted command output.
